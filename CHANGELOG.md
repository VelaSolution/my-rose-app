- 근무 스케줄 (주간 시프트 그리드, 이전 주 복사)
- 데일리 체크인 (스탠드업, 연속 체크인 스트릭)

**재무 관리 페이지**

**기존 탭 고도화**
- 게시판: 게시글 수정/삭제 + 검색 + 정렬(최신/인기/댓글순)
- 급여: 수정/삭제 + PDF 출력
- 태스크: 다중 필터(우선순위/상태/담당자) + 검색
- 칭찬: 카테고리 6종 + 받은/보낸 통계 + 24시간 내 삭제
- 결재: 다단계 결재선 (A→B→C 순차 승인, 스텝 시각화)
- 위키: 문서 권한 제어 + 수정 이력 조회
- 근태: 직원 검색 + 기간 필터 + 건수 표시
- 보고서: 일일/이슈/프로젝트 PDF 출력
- 휴가: 팀 휴가 중복 경고 + 승인 코멘트
- 목표: 마일스톤/하위 목표 + 자동 진행도
- 공지: 예약 발행 + 읽음율 분석
- 주소록: vCard 내보내기 (개별/전체)
- 대시보드: 체크인/자원예약/CRM 위젯 추가
