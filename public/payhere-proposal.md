|------|------|------------|
| **수익 시뮬레이터** | 좌석·객단가·비용 입력 → 순이익·손익분기점 자동 계산 | 매장이 돈 버는 구조인지 즉시 판단 |
| **매장 가계부** | 일별 수입/지출 기록 → 카테고리별 분석 + 월간 순이익 | 매일 2분, 돈 새는 곳 파악 |
| **메뉴 원가 계산기** | 식재료별 원가 입력 → 메뉴당 순이익 계산 | 가격 인상 판단 근거 |
| **인건비 계산기** | 시급·근무시간 → 주휴수당·4대보험 포함 실제 인건비 | 채용 전 비용 예측 |

| **HQ 인트라넷** | 출퇴근·보고서·채팅·결재·파일 관리 | 직원 관리 원스톱 |

## 2. PayHere 연동 시너지

### 현재 구조의 한계
```
[사장님] → 수동 입력 → [VELA 분석] → 인사이트
              ↑
         여기서 이탈 발생
```

### PayHere 연동 후
```
[PayHere POS] → 자동 수집 → [VELA 분석] → 인사이트
                                ↑
                         이탈 제로, 리텐션 확보
```

### 구체적 시너지

#### 2-1. PayHere가 얻는 것
| 가치 | 설명 |
|------|------|
| **가맹점 리텐션** | POS만으로는 "결제 도구"지만, VELA 붙으면 "경영 파트너"가 됨 |
| **ARPU 상승** | VELA 프로 플랜(29,900원/월)을 번들로 제공 시 업셀 |
| **차별화** | 카카오페이/네이버페이에 없는 "경영 분석" 기능 |
| **데이터 인사이트** | 가맹점 수익 구조 파악 → 리스크 스코어링, 대출 심사 활용 |
| **B2B SaaS 확장** | HQ 인트라넷으로 중규모 프랜차이즈 시장 진입 |

#### 2-2. VELA가 얻는 것
| 가치 | 설명 |
|------|------|
| **자동 데이터** | POS 매출 자동 연동 → 수동 입력 제거 → 리텐션 핵심 해결 |
| **사용자 확보** | PayHere 가맹점 → VELA 즉시 유입 |
| **결제 채널** | PayHere 정산에서 VELA 구독료 자동 차감 가능 |

#### 2-3. 가맹점(사장님)이 얻는 것
| 가치 | 설명 |
|------|------|
| **카드매출 자동 입력** | 가계부에 카드매출 자동 반영, 현금+지출만 직접 입력 |
| **수수료 포함 순이익** | PayHere 수수료 차감 후 "진짜 남는 돈" 확인 |
| **정산 현황** | 정산 지연, 분쟁, 환불 알림 |
| **원클릭 리포트** | 세무사에게 보낼 매출 리포트 자동 생성 |

## 3. 통합 방식 제안

### Option A: API 연동 (파트너십)
- PayHere 대시보드에 VELA 위젯 임베드
- PayHere 정산 데이터 → VELA API 전송
- 수익 분배: PayHere 70% / VELA 30%
- 소요 기간: 4~6주

### Option B: 화이트라벨
- VELA 기능을 "PayHere Analytics"로 리브랜딩
- PayHere 대시보드 내 네이티브 통합
- 라이선싱 월정액 또는 가맹점당 과금
- 소요 기간: 8~12주

### Option C: 인수 (Acqui-hire)
- VELA 전체 기술 자산 + 코드베이스 이전
- 개발팀 합류 (기술 이전 + 유지보수)
- 일시금 또는 일시금 + 어닝아웃
- 소요 기간: 즉시 (기술 이전 2~4주)

### 권장: Option A로 시작 → 검증 후 B 또는 C 전환
- 리스크 최소화: 먼저 API 연동으로 시장 반응 검증
- 지표 확인: 가맹점 활성화율, 이탈률 감소, ARPU 변화
- 데이터 기반 의사결정 후 심화 통합

## 6. 기존 API (즉시 활용 가능)

VELA는 이미 외부 연동용 REST API를 제공합니다:

**엔드포인트**: `POST /api/v1/analyze`
**인증**: Bearer Token
**입력**: 업종, 좌석, 객단가, 회전율, 월세, 인건비, 배달 비율
**출력**: 순이익, 마진율, 비용 구조, 벤치마크 비교, 12개월 시뮬레이션, AI 전략

→ PayHere 대시보드에서 이 API를 호출하면 **즉시** 경영 분석 위젯 구현 가능

**API 문서**: `/api/v1/docs`

## 7. 왜 지금인가

1. **외식업 SaaS 시장 성장**: 코로나 이후 디지털 전환 가속, 사장님들의 도구 수용도 상승
2. **POS 시장 경쟁 심화**: 결제 수수료 경쟁은 한계 → 부가가치 서비스로 차별화 필수
3. **AI 도구 대중화**: ChatGPT 이후 AI 기능에 대한 거부감 감소, "AI가 해주는" 것에 대한 기대치 상승
4. **VELA 기술 완성도**: 88페이지, 36개 도구, 112개 테스트 통과 — 즉시 통합 가능한 상태
