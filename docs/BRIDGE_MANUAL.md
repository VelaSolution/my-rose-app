## 1. 시작하기

### 로그인
- 관리자가 등록한 이메일로 로그인합니다.
- 최초 로그인 시 **승인 대기** 상태이며, 대표가 승인하면 모든 기능을 사용할 수 있습니다.

### 화면 구성
| 영역 | 설명 |
|------|------|
| **사이드바** (좌측) | 탭 메뉴 — 홈, 협업, 소통, 경영 그룹으로 구분 |
| **상단 바** | 검색(Ctrl+K), 알림, 다크모드 토글 |
| **본문 영역** | 선택한 탭의 콘텐츠 |
| **모바일** | 하단 플로팅 메뉴 버튼으로 주요 탭 이동 |

### 다크모드
상단 바의 토글 버튼으로 켜고 끕니다. 설정은 자동 저장됩니다.

## 2. 홈 (대시보드)

로그인하면 가장 먼저 보이는 화면입니다.

### 퀵 액세스
화면 상단에 8개 바로가기 버튼이 있습니다:
- 출퇴근 / 업무 / 결재 / 채팅 / 공지 / 메일 / 일정·예약 / 문서

### 위젯
대시보드에는 다양한 정보 위젯이 표시됩니다:
- **핵심 지표** — 사용자 수, 매출, 구독, 가입자
- **오늘의 업무** — 나에게 배정된 태스크
- **결재 & 출퇴근** — 대기 중인 결재, 출근 상태
- **팀 상태** — 팀원 접속 현황
- **최근 공지** — 새 공지사항
- **예정 일정** — 다가오는 일정
- **CRM 파이프라인** — 영업 현황

### 위젯 편집
우측 상단 **⚙ 편집** 버튼을 누르면:
- 위젯 순서를 드래그로 변경
- 불필요한 위젯 숨기기/보이기
- **✓ 완료** 버튼으로 저장

## 3. 메일

사내 메일을 주고받는 기능입니다.

### 폴더 구성
| 폴더 | 설명 |
|------|------|
| 받은편지함 | 수신 메일 |
| 보낸편지함 | 발신 메일 |
| 중요 | 별표 표시한 메일 |
| 임시보관함 | 작성 중인 임시저장 |
| 휴지통 | 삭제한 메일 |

커스텀 폴더를 추가로 만들 수 있습니다.

### 메일 보내기
1. **새 메일** 버튼 클릭
2. 수신자 선택 (팀원 검색 가능)
3. 제목, 본문 작성
4. 첨부파일 추가 (선택)
5. **보내기** 클릭

### 부가 기능
- **읽음 확인** — 수신자가 읽으면 ✓✓ 파란색 표시
- **자동분류** — 발신자/제목/본문 조건으로 폴더 자동 이동
- **부재중 자동응답** — 기간 설정 후 자동 회신
- **별표(중요)** — 메일 목록에서 ☆ 클릭

## 4. 근태관리

### 출퇴근
**출근 체크인 방법:**
1. 근태관리 탭 접속
2. **출근** 버튼 클릭
3. 메모 입력 (선택)
4. 자동으로 시간 기록

- GPS 기반 위치 확인이 설정된 경우, 사무실 반경 내에서만 출근 가능
- 09:00 이후 출근 시 **지각** 자동 처리
- 퇴근 시 **퇴근** 버튼 클릭

### 근태 현황
- 일별 출퇴근 기록 확인
- 주간/월간 캘린더 뷰
- 상태: 정상 / 지각 / 조퇴 / 결근 / 휴가 / 출장
- 초과근무 시간 자동 계산

### 휴가 신청
1. **휴가** 서브탭 클릭
2. 휴가 종류 선택:
   - 연차 / 반차(오전) / 반차(오후) / 병가 / 경조 / 출장 / 기타
3. 시작일~종료일 선택
4. 사유 입력
5. **신청** 클릭 → 승인 대기 상태

- 반차는 자동으로 0.5일 계산
- 승인 상태: 대기 → 승인 또는 반려

## 5. 업무

태스크를 등록하고 관리하는 탭입니다.

### 서브탭
| 탭 | 설명 |
|-----|------|
| **태스크** | 리스트 형태로 업무 관리 |
| **칸반** | 보드 형태 (드래그로 상태 변경) |
| **일정** | 캘린더에서 마감일 확인 |
| **간트** | 타임라인 차트로 프로젝트 일정 |
| **체크아웃** | 데일리 업무 완료 체크 |

### 태스크 만들기
1. **+ 태스크** 버튼 클릭
2. 입력 항목:
   - 제목
   - 담당자 배정
   - 마감일
   - 우선순위 (긴급 / 높음 / 보통 / 낮음)
   - 진행률 (0~100%)
   - 연결할 목표 (선택)
3. **등록** 클릭

### 상태 관리
태스크 상태는 4단계로 구분됩니다:
1. **대기** — 아직 시작 안 함
2. **진행중** — 작업 중
3. **검토** — 리뷰 대기
4. **완료** — 작업 완료

칸반 보드에서는 드래그 앤 드롭으로 상태를 변경할 수 있습니다.

### 댓글
각 태스크에 댓글을 달아 팀원과 소통할 수 있습니다.

## 6. 게시판

### 서브탭
| 탭 | 설명 |
|-----|------|
| **게시판** | 자유 게시판, 질문, 정보 공유 |
| **공지** | 공식 공지사항 |
| **칭찬** | 동료 칭찬하기 |
| **설문** | 설문조사 |

### 게시판
- **카테고리:** 자유 / 공지 / 질문 / 정보 / 부서 / 버그 / 건의
- 글 작성, 댓글, 좋아요 기능
- 버그 카테고리는 상태(접수/진행/완료/보류)와 우선순위 추적 가능
- 검색 및 정렬 (최신순 / 인기순 / 댓글순)

### 공지사항
- 팀장 이상만 작성 가능
- 카테고리: 일반 / 긴급 / 인사 / 경영
- **중요 공지**로 설정하면 로그인 시 팝업으로 표시
- 읽음 수 추적

### 칭찬 (Kudos)
동료에게 칭찬을 보내는 기능입니다:
1. 대상 팀원 선택
2. 카테고리 선택 (업무성과 / 팀워크 / 리더십 / 창의성 / 도움 / 기타)
3. 이모지 선택 (👏 🌟 💪 🔥 ❤️ 🎉 🏆 💎)
4. 칭찬 메시지 작성
5. **보내기** 클릭

- 월간 칭찬 랭킹 확인 가능

### 설문
- 질문 유형: 단일선택 / 복수선택 / 서술형 / 평점
- 마감일 설정
- 응답 결과 요약 확인

- 🟠 휴가

## 8. 문서관리

### 파일 관리
- 폴더 생성 및 계층 구조
- 파일 업로드 (드래그 앤 드롭 지원)
- 파일 작업: 이름변경, 이동, 복제, 삭제
- **버전 관리** — 파일 수정 이력 추적
- **보안 등급** — 공개 / 내부용 / 기밀 / 극비
- **색상 태그** — 파일에 컬러 태그 지정
- **즐겨찾기** — 자주 쓰는 파일 별표
- **공유 링크** — 외부 공유용 링크 생성 (만료일 설정 가능)
- 보기 모드: 그리드 / 리스트
- 저장 용량: 10GB

### 위키
사내 지식을 문서화하는 공간입니다:
- 문서 작성 (마크다운 지원)
- 카테고리별 분류 (가이드 / 정책 / 기술 / 일반)
- 태그 기능
- 자동 목차 생성
- 조회수 추적
- 검색 기능

## 9. 전자결재

### 결재
문서를 작성하고 결재를 요청하는 기능입니다.

**기본 제공 양식 (14종):**

| 양식 | 용도 |
|------|------|
| 휴가신청서 | 연차/반차/병가 등 |
| 지출결의서 | 비용 지출 승인 |
| 출장신청서 | 출장 계획 승인 |
| 구매요청서 | 물품/서비스 구매 |
| 업무협조전 | 부서 간 협조 요청 |
| 시말서 | 경위 보고 |
| 경조사신청서 | 경조금 신청 |
| 회의록 | 회의 내용 기록 |
| 교육신청서 | 교육/세미나 참가 |
| 기안서 | 기획안 제출 |
| 일일보고서 | 일일 업무 보고 |
| 주간보고서 | 주간 업무 보고 |
| 월간보고서 | 월간 업무 보고 |
| 자유양식 | 기타 문서 |

**결재 흐름:**
1. 양식 선택 → 내용 작성
2. 결재자 지정
3. **제출** → 대기 상태
4. 결재자가 **승인** 또는 **반려**
5. 긴급 결재 플래그 가능

### 보고서
- 일일/주간/월간 보고서 작성
- 이슈 보고서 (문제 추적)
- 프로젝트 보고서 (진행률 관리)
- 보고서에서 결재 문서 자동 생성 가능

## 10. 경리관리

### 경비/고정비

#### 지출 등록
1. **+ 경비 등록** 버튼 클릭
2. 입력 항목:
   - 날짜
   - 카테고리 (식비 / 교통비 / 사무용품 / 마케팅 / 소프트웨어 / 통신비 / 복리후생 / 기타)
   - 금액 + 통화 선택 (KRW / USD)
   - 결제수단 (법인카드 / 개인카드 / 사업자카드 / 현금 / 계좌이체)
   - 설명, 비고
3. **등록** 클릭 → 승인 대기 상태

#### 현황 탭
- 월 고정비 / 이번달 지출 / 월 총 비용 / 연간 예상 한눈에 확인
- 카테고리별 지출 비율 차트

#### 고정비 관리 (대표/이사만)
매월 반복되는 비용을 등록합니다:
- 항목명 (예: Vercel Pro, .com 도메인)
- 카테고리 / 금액 / 결제 주기 (월/분기/반기/연) / 납부일
- 활성/비활성 토글
- 자동 월 환산 계산

#### 엑셀 내보내기
- 지출 목록을 선택하여 엑셀 파일로 다운로드
- 전체 또는 선택한 건만 내보내기 가능

### 급여
- 대표/이사가 월별 급여명세서 등록
- 팀원은 본인 급여만 확인 가능
- 항목: 기본급 / 초과근무 / 상여 / 4대보험 / 소득세 / 실수령액

## 11. 인사관리

### 팀원 관리
- 팀원 추가 (이름, 직무, 이메일, 권한)
- 권한 변경 (대표만 가능)
- 승인/승인취소
- 상태 변경 (활동중/자리비움/오프라인)
- 팀원 삭제

### 인사 발령 이력
인사 변동을 기록하고 추적합니다:
- 유형: 입사 / 퇴사 / 승진 / 부서이동 / 직책변경
- 타임라인 형태로 이력 확인
- 팀원/유형별 필터
- 수동 발령 등록 가능 (대표/이사)

### 주소록
팀원 연락처를 관리하는 디렉토리입니다:
- 이름, 부서, 직책, 이메일
- 전화번호, 내선번호, 연락처(휴대폰), 집주소
- 카드 뷰 / 조직도 뷰 전환
- 검색 및 부서 필터
- **명함 기능:**
  - 명함 보기 — 팀원의 명함 카드 확인
  - 내 명함 편집 — 회사명, 직책, 연락처, 주소, 좌우명 등 커스터마이징
  - 명함 공유 — 클립보드 복사
- **연락처 내보내기** — VCF(vCard) 파일로 개별/전체 내보내기

### 조직도
부서별 조직 구조를 시각적으로 확인합니다.

## 12. 예약

### 예약
회의실, 차량, 장비를 예약합니다:
1. 자원 선택 (회의실 / 차량 / 장비)
2. 날짜, 시간대 선택 (08:00~20:00)
3. 목적 입력
4. 참석자 추가 (선택)
5. **예약** 클릭

- 시간 충돌 자동 감지
- 내 예약 목록 확인
- 예약 상태: 예약 / 사용중 / 취소 / 완료

### 자산 관리
회사 자산(노트북, 모니터 등)을 등록하고 추적합니다:
- 자산 등록 (이름, 카테고리, 시리얼번호, 구매일, 구매가)
- 상태: 사용중 / 보관중 / 수리중 / 폐기
- 담당자 배정
- 대여/반납 이력

### 방문자 관리
외부 방문자를 등록하고 관리합니다:
- 방문자 정보 (이름, 소속, 목적, 연락처, 차량번호)
- 담당자(호스트) 지정
- 방문 예약 → 도착 체크인 → 퇴실 체크아웃
- 상태: 예약 / 방문중 / 퇴실

### 차량 운행일지
차량 사용 기록을 관리합니다:
- 날짜, 운전자, 차량명
- 출발지/도착지, 출발/도착 km
- 주행거리 자동 계산
- 주유량, 주유비 기록

## 13. 성과관리

### KPI
핵심 성과 지표를 추적합니다:
- 매출, 사용자 수, 전환율, 이익
- 날짜별 기록 추가
- 트렌드 분석 및 비교

### 목표
조직 목표를 설정하고 관리합니다:
- 목표 생성 (제목, 목표값, 기간, 측정 유형)
- 마일스톤 하위 목표 설정
- 진행률 시각화 (프로그레스 바)
- 상태: 진행중 / 완료 / 실패
- 인당 최대 2개 활성 목표

### 인사평가
성과 평가를 진행합니다:
- 평가 기간 설정
- 평가 유형: 자기평가 / 상사평가 / 동료평가
- 목표 달성도 + 역량 점수
- 코멘트 작성
- 제출 → 확정 흐름

## 16. 단축키 & 팁

### 단축키
| 키 | 동작 |
|----|------|
| `Ctrl + K` (Mac: `Cmd + K`) | 전체 검색 열기 |
| 로고 클릭 | 대시보드로 이동 |

### 유용한 팁
- **대시보드 위젯 커스터마이징** — ⚙ 편집 버튼으로 원하는 위젯만 표시
- **알림 뱃지** — 사이드바 탭 옆 숫자로 미확인 항목 빠르게 파악
- **엑셀 내보내기** — 경비 지출 내역 선택 후 한 번에 다운로드
- **명함 공유** — 주소록에서 명함 보기 → 명함 공유로 텍스트 복사
- **VCF 내보내기** — 주소록 연락처를 핸드폰에 바로 저장 가능
- **칸반 보드** — 업무 상태를 드래그로 빠르게 변경
- **다크모드** — 야간 작업 시 눈 피로 감소
